# focus
A simple CLI tool to help block distracting websites at a DNS level.

Developed with Claude 🦀

# Installation

```bash
# assuming you are running this from the project directory
# and that ~/.local/bin is on PATH
$ ln -s $(pwd)/focus ~/.local/bin/focus
```

# Notes

Chrome will cache DNS, but this can be cleared from `chrome://net-internals/#dns`
